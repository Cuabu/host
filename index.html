Descargar herramientas

Ver el Administrador de la Base de Datos Ir a otra página

Antes de utilizar esta herramienta, se recomienda instalar la versión de XAMPP o LAMP según su sistema operativo. Luego, deberá descargar el archivo de la base de datos que se encuentra en el botón de esta página. Después, acceda a phpMyAdmin desde un navegador web y importe el archivo que contiene la base de datos.
